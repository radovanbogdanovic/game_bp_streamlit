# Relaciona Algebra Drill

Streamlit quiz app for practicing relational algebra terms through shuffled definition prompts.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud

1. Push this folder as its own Git repository to GitHub.
2. In Streamlit Cloud, choose this repo.
3. Set the main file path to `app.py`.
